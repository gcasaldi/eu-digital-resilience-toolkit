# EU Digital Resilience Toolkit

Interactive compliance assessment tool for NIS2 and DORA regulations.

**Author**: Giulia Casaldi  
**License**: MIT  
**Python**: 3.8+  
**Framework**: Streamlit

## Features

- ✅ NIS2 Audit Log Readiness assessment
- ✅ DORA Third-Party Risk Assessment
- ✅ Automated risk scoring (0-100)
- ✅ PDF report generation
- ✅ Optional email sharing (GDPR-compliant)
- ✅ Step-by-step guided workflow

## Technologies

- **Frontend/Backend**: Streamlit
- **PDF Generation**: FPDF2
- **Email**: Python smtplib (SMTP/TLS)
